Conditionals

Python supports logical conditions:

  1. Equals: a == b
  2. Not Equals: a != b
  3. Less than: a < b
  4. Less than or equal: a <= b
  5. Greater than: a > b
  6. Greater than or equal: a >= b

Loops

In Python, there are two types of loops:

  1. for loop
  2. while loop

for loop

Used for iterating over a sequence:

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)

while loop

Executes a block while a condition is true:

i = 1
while i < 6:
    print(i)
    i += 1
